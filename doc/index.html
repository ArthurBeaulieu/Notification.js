src/js/Notification.js

Notification.js is part of the MessCss web framework. It can both work stand alone and in MessCss. The class itself is pretty easy to use and can be set up in a minute, but also gives the possibility to tweak it a lot, using options. -- KISS (Keep It Simple, Stupid!)

There are four different types of notifications to give your users different feedbacks: Succes, Info, Warning and Error. These are highly customizable using options object. Also, the CSS style is made in SCSS, so you can tweak the variable to make this class perfectly fit your needs.

Examples


Notification handler

The easiest way to implement this class is to attach an instance to the window DOM object (1), so you can access it from anywhere in your app.
Optionnaly, you can pass an options object to tune the handler behavior (2).
Warning: transition time must not exceed half the duration (for obvious, logical questions), otherwise, it will be resetted to the default value.

// (1) - Default options (see class documentation)
window.notification = new Notification();

// (2) - Custom notification handler
window.notification = new Notification({
   position:    'top-right', // Notification handler position in viewpoint // TODO add top-center, bottom center
   thickBorder: 'top',      // Global notification thick border location
   duration:    5000,        // Total duration of a notification on screen in milliseconds
   transition:  200,         // Fade in and out transition duration in milliseconds
   maxActive:   10            // The max number of simultaneously opened notification
});


Notification type

Once attached to the DOM, add a notification from anywhere in your app logic (3).
You can also call a new typed notification (4) if you find it more suitable for your need.
Note: type and message are the only mandatory options (no need to put type when using example (3), since they are already typed).
Also, storing the notification ID is only useful if you intend to manually dismiss the notification later on.

// (3) - Simple notification type abstraction
let ids = window.notification.success({ message: 'Great success!' });
let idi = window.notification.info({ message: 'Much info, very text' });
let idw = window.notification.warning({ message: 'Snake?' });
let ide = window.notification.error({ message: 'Snaaaaaaaake!' });

// (4) - Custom notification construction with all possible parameters
let id = window.notification.new({
   type:        'info',                         // Mandatory, Notification type
   title:       'My test notification',         // Notification head title
   message:     'Lorem ipsum dolor sit amet.',  // Mandatory, Notification message
	  duration:    5000,						   // Notification duration
	  iconless:    false,						   // Icon in the notification
   thickBorder: 'top'                           // Local notification thick border location
   closable:    true,                           // Ability for user to close notification
   sticky:      false,                           // Notification will dim instead of close
   renderTo:    document.body,                  // Render the notification anywhere you want
   CBtitle:     'My callback',                  // Callback button text
   callback:    function() {                    // Callback function
       alert('Called from my notification');
   }
});


Archived notifications

When a notification reaches the end of its lifecycle and disapear, it is still store sorted in an history array.
In this array, you have access to all notification in a chronological order (index 0 is the oldest), their attributes and their timestamps (creation, deletion, duration on screen).

let historyLength = window.notification.getHistoryLength(); // Returns the number of archived notifications
let history       = window.notification.getHistory();       // Returns a sorted array of archived notification


Notification manual destruction

Finally, you can manually dismiss a notification using its ID or dismill them all:

window.notification.dismiss(id);
window.notification.dismissAll();
